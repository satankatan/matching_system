Техническое задание на проект:
Реализовать веб-приложение, которое использует NLP.
Проект	предусматривает	разработку	компьютерного	приложения, позволяющего решить имеющиеся проблемы в игровой индустрии. Необходимо разработать программу для ПК, которая будет подбирать игрока по предпочтениям пользователя (навыки игрока, уровень, пол, рейтинг в игре и другое). Результатом выполнения проекта является веб-приложение
«Teammate Gaming Search», позволяющее решить проблему поиска игроков для совместной игры.
Требования к проекту:
К	завершению проекта	готовое	приложение	должно включать в себя:
–	использование NLP для улучшения алгоритмов подбора тиммейтов и повышения качества рекомендаций;
–	визуализация	данных	о	потенциальных	союзниках	и	их соответствие заданным критериям;
–	проектирование системы веб-приложения;
–	разработку	веб-приложения	для	поиска	тиммейтов	в многопользовательских играx.

Платформа «Teammate Gaming Search» подразумевает, что пользователи могут сократить время для поиска тиммейтов в команду для конкретной игры. При регистрации у пользователя есть возможность выбрать из предложенных игр те, которые ему предпочтительнее всего.
Геймеры будут создавать профили с возможностью написать краткое резюме о себе, указывая свои предпочтения по жанрам игр, уровню, опыту и времени, которое они могут уделить играм. Пользователи смогут просмотреть профили других пользователей и свайпать вправо, чтобы добавить их в список друзей. Игроки главным образом будут подбираться по игре, которую они выберут, далее будут отбираться пользователи по рейтингу, следующими уже будут отбираться такие данные как время, проведённое в игре, позиция на игровой карте (если керри то подбираем саппорта). Для выполнения этой задачи будет использоваться модель нейронной сети.
Также у геймеров будет возможность делиться своими игровыми достижениями, стратегиями и советами с другими участниками сообщества. Это могут быть посты и видео. У пользователей будет возможность общаться между собой, обсуждать тактику и планировать игровые сессии через встроенный чатовый интерфейс
Стэк технологий: Python, Flask, JS, CSS, HTML, SQLite3
